Book File Upload Instructions

You have two options to submit your file:

1. Upload File Link (Recommended)

  • Use a cloud storage service such as Google Drive, OneDrive, Dropbox, or Mega.
  • Ensure the link is publicly accessible or shared with the appropriate permissions.
  • Copy and paste the direct link in the provided field.

2. Upload Directly

  • You can upload the file directly to our platform.
  • File Format: Only .pdf or .doc files are accepted.
  • Maximum File Size: 50MB.
  • Ensure that your document is properly formatted and complete before uploading.

Important Notes:

• The recommended method is uploading via a cloud storage link to avoid file size limitations.

• If uploading directly, verify file does not exceed 50MB and is in the correct format.

• If you experience any issues, please contact our support team.
